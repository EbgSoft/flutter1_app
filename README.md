# Flutter Ders: 1

Merhaba öncelikle Flutter ile  basit uygulama oluşturalım.

Uygulama giriş fonksiyonu ve bir stateless bir widget oluşturalım.

Sayfanın ortasında yazı yazdıralım ve bunu renklendirelim.
